# Data Structures & Algorithms (DSA)

This section focuses on building strong problem-solving and algorithmic thinking skills through hands-on implementation.

The goal is to develop confidence in identifying problem patterns, selecting appropriate data structures, and designing efficient algorithms by solving progressively challenging problems.

## Topics

* Arrays
* Strings
* Hash Maps
* Two Pointers
* Sliding Window
* Stack
* Queue
* Linked Lists
* Binary Search
* Trees
* Graphs
* Heaps
* Recursion
* Backtracking
* Greedy Algorithms
* Dynamic Programming
* Bit Manipulation
